# Grizzly Adventures
Source code and assets for Grizzly Adventures game, famous 2D platformer game from 2000's

Tile engine, colisions, Animated sprites, Audio, Game logic


PC, MacOS, iOS

Requirement : Lib glfw3 https://www.glfw.org/download
